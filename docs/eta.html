8.1 Setup

We start by defining a set of ETAs to use in the plots.

etas <- c(
  "ETA1//ETA-CL", 
  "ETA2//ETA-V2", 
  "ETA3//ETA-KA"
)

This is in the col-label format described earlier. We also set out a set of covariates that we can use for ETA diagnostics.

covs <- c(
  "WT//Weight (kg)", 
  "ALB//Albumin (g/dL)", 
  "SCR//Creatinine (mg/dL)"
)
